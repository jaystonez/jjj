== Description ==
Simple way to import or generate user name, password, email id and many more for wordpress using this plugin

Through this plugin user can able to,
*	Generate new users in wordpress
*	Import new users from csv file

* Updated to 3.8.1

v 1.0

== Installation ==

1. Upload the "user-generator" directory to the plugins directory.
2. Go to the plugins setting page and activate "User Generator"
